그룹스를 선택하는 이유

All-In-One 디지털 시대의 교육솔루션

디지털 시대, 교육은 어떤 모습을 꿈꾸고 있나요?
언제 어느곳에서나 학습 가능한 그룹스로 학습자들끼리 의견을 공유하며 협업해나가고, 각종 통계 기능을 통해 학습 데이터를 분석하여 더 나은 콘텐츠를 설계, 제작, 배포할 수 있는 교육 시스템, 상상만 했던 것들을 모두 준비하였습니다.

  • Learning Everywher
    언제 어디서나 학습

    필요할때 언제든 꺼내보는 학습노트로 언제 어디서나 학습관리가 가능합니다.

  • Spot Learning
    이벤트(즉각적) 학습

    그룹스는 쉽고 간편하게 콘텐츠를 만들어 빠르게 콘텐츠를 배포하고 학습할 수 있습니다.

  • Learning Analytics
    학습 분석

    진도율, 수료여부로 이루어진 학습통계에서 벗어나 실시간 학습 현황, 인기 매터리얼, 학습자 랭킹 등을 조회해 보십시오.
